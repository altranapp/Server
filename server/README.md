# 🚀 Crypto Investment Platform

A full-stack crypto investment simulation platform built with:

- Node.js
- Express.js
- MongoDB
- JWT Authentication

---

## ✨ Features

### 👤 User
- Register & Login
- JWT Authentication
- Dashboard with balance
- Select trading tiers (1, 2, 3)
- Request deposits & withdrawals
- Daily profit system
- Transaction history

### 👑 Admin
- View all users
- View all transactions
- Approve / Reject deposits
- Approve / Reject withdrawals
